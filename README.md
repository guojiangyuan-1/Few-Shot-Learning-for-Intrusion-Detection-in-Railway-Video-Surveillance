# Few-Shot-Learning-for-Intrusion-Detection-in-Railway-Video-Surveillance
这是一个基于小样本学习的用于铁路入侵检测的项目，代码基于PaddlePaddle实现。
  深度学习模型在实际铁路部署中具有场景差异问题及侵限小样本问题，这两个问题导致现有传统深度学习模型在实际铁路视频监控场景中迁移困难、样本稀少![image](https://user-images.githubusercontent.com/82765110/228853481-64fbdccc-e4f9-4505-96eb-d15ce687ccab.png)
